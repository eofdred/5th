# Wiki Pop!

A playful word-discovery interface built from the June 2026 Simple English Wikipedia Kiwix snapshot.

The included compact dataset contains 1,200 articles and 4,317 sentences extracted from `wikipedia_en-simple_all_mini_2026-06.zim`. The original archive is not included or modified.

Open `index.html` through a local web server, or visit:

https://muratdogusan.com/projects/wiki-pop/
