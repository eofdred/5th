# **App Name**: Vet-VX

## Core Features:

- Topic List Display: Display a list of key farm safety topics on the main page.
- Topic Navigation: Each topic on the main page should link to its own dedicated page with detailed information.
- Comprehensive Topic Coverage: Include topics: Quarantine, Disease monitoring, Feed safety, Staff hygiene, Teat cleaning, Disinfection at the entrance, Dispose of dead animals, Animal handling order, Other domestic or wild animals in contact, Record keeping

## Style Guidelines:

- Primary color: Soft green (#A7D1AB) to evoke feelings of health and safety.
- Background color: Very light, desaturated green (#F0FAF2).
- Accent color: Warm yellow-orange (#E3B577) to draw attention to important elements.
- Body and headline font: 'PT Sans', a humanist sans-serif, to ensure good readability and a modern feel.
- Use simple, clear icons to represent each safety topic.
- Maintain a clean and intuitive layout for easy navigation.
- Subtle animations, such as a gentle fade-in effect on topic items as they load.
